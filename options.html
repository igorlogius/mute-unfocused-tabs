Manual Mode:

Set inital states (via. RegEx-URL-Matches)

  • Default Mode: Matching tabs will initally be unmanaged
  • Manual Mode: Matching tabs will initally be managed
  • Example 1: ^https:\/\/www\.twich\.tv.*, matches all twitch.tv sites
  • Example 2: ^https:\/\/www\.youtube\.com.*, matches all youtube.com sites
